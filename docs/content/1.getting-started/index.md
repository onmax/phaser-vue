Start here when you want the shortest path from install to a running canvas. These pages cover the shared mental model first, then the runtime-specific setup.

The core rule stays the same across Vue and Nuxt: `PhaserGame` owns the actual `Phaser.Game` instance, and Phaser only boots on the client.
